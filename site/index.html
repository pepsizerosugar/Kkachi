3 tabs tucked away

Native macOS menu-bar utility

Close idle tabs without making closing feel final.

Kkachi watches supported browsers locally, closes tabs that have gone quiet, and keeps a small restore history so each tab can come back.

What happens

A quiet loop for tabs you are not using.

Watch

Read only the tab address and title.

Kkachi uses macOS Apple Events and avoids page content, form state, cookies, and scripts.

Prune

Close only tabs that pass the policy.

Active, protected, ambiguous, or unavailable tabs are skipped instead of guessed.

Restore

Keep the path back short.

The menu keeps a capped local history for recent pruned tabs and routes restore to the browser.

Privacy model

The promise is deliberately narrow.

Kkachi is built around a smaller claim that can be checked in source: it stores only the minimum restorable context and keeps it on your Mac.
